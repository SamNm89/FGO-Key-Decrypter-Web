# FGO Key Decryptor Web

A simple, secure, client-side web tool to decrypt FGO authentication files and extract the User ID, Auth Key, and Secret Key.

## How to Host on GitHub Pages

1.  **Push to GitHub**:
    *   Initialize a git repository in this folder (or the parent).
    *   Commit `index.html` and `script.js`.
    *   Push to your GitHub account.

2.  **Enable Pages**:
    *   Go to your Repository Settings > **Pages**.
    *   Under "Build and deployment", select **Source** as "Deploy from a branch".
    *   Select your branch (e.g., `main` or `master`) and the folder `/` (root).
    *   Click **Save**.

3.  **Access**:
    *   Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Usage
1.  Navigate to the website.
2.  Drag and drop your `54cc790bf952ea710ed7e8be08049531` file.
3.  The tool will decrypt it locally (in your browser) and display your keys.
4.  Click "Download .txt Report" to save them safely.

## Privacy
This tool runs entirely in your browser. No files or keys are uploaded to any server.
